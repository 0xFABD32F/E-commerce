# Project Structure

## Pages Overview

### `Pages/login`
Handles user authentication and account management using JWT (JSON Web Tokens).
- Login functionality
- Logout functionality  
- Account creation

### `Pages/Cart.cshtml.cs`
Contains all cart management logic for handling shopping cart operations.

### `Pages/Index.cshtml.cs`
The application homepage and main entry point.

### `Pages/produit`
Manages product-related operations to supply the database:
- Create products
- Read product details
- Update product information
- Delete products

### `Pages/user`
Handles user account operations:
- CRUD operations for user data
- Update account details
- *Note: Will be integrated with `Pages/login/Details` in future updates*
